🖥️ Self-Host Your Own Node

Run Music Bank on your own hardware. Unlimited storage. Full control. No platform fees.

1

Clone the Repository

git clone https://github.com/TheInnerI/music-bank.git
2

Configure Environment

cp .env.example .env && nano .env
3

Deploy

docker-compose up -d
4

Access Your Node

http://localhost:8090

Minimum Requirements

  • Linux (Ubuntu 22.04+) or macOS
  • 2 GB RAM (4 GB recommended)
  • Docker + Docker Compose
  • 10 GB disk space (code + OS)
  • Your own storage for audio files (local disk, NAS, or cloud)
